Stretch Questions

    1. What is the difference between an inline element and a block element?
        Inline elements do not start at a new line and takes up as much space as it needs where as a block element starts on a new line and takes up all available width.

    2. What happens when an element is positioned absolutely? 
        An element is taken out of the normal document flow and it's position is set to the upper left corner of the parent div.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
        display: inline-block
    
    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
        Section, div, and p are all display blocks by default, image and span are inline by defalt, and button is an inline-block.
    
    5. In your own words, explain the box model. What is the fix for the box model? 
        The box model is the wrappings around the content of an element using padding, border, and margin to establish the content's given space on the webpage. You use box-sizing: border-box: to set the sizing for your box.
